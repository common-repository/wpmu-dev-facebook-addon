== Description ==

An addon on WPMU Ultimate Facebook, changes the way OpenGraph tags create Descrition (make it according to Simple Facebook Connect). Plus if you are using Infinite SEO it will use the same description for it.

== Installation ==

1. Download
2. Upload to your `/wp-contents/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
